This is a small project I made myself. Here some information about it:

  • Frontend: It is made with ReactJS v18.1.0. Implementing useContext and useReducer for the administration of states.

  • Backend: It is made using NodeJS with express framework.

  • For the data base I used mongoDB with mongoose ODM.

Description: First you will see a LogIn form, with the option to go to a SignUp form if you dont have an account. All the users are storaged at the DB and in the API I have implemented JsonWebToken for the authentications. Once you LogIn, you will se the main page, were you can create and administrate tasks that are related to your account and only you have the permissions to modify them. If you LogOut and LogIn again you will see the projects and tasks as you left them.
